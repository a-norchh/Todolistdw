### Todo App

todolist work with json api server.

main feature
- Add Todo
- Edit Todo
- Delete todo
- Show what's done
- Mark as Done
- Show Progress

Using Raeact base, SCSS, React Hooks (useState, useEffect), React Context API

After you pull the project // you have to start json server first.
- npm install
- npm run start
